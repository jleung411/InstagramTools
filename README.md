# InstagramTools

### config.json format

~~~~
{"log_level":"debug",
"device":"<device_name>",
"followers":"followers.json",
"excludes":"excludes.json",
"messages":"messages.json",
"username":"<username>",
"password":"<password>"}
~~~~

### Dependencies

* npm install --save winston
* npm install --save bluebird
* npm install --save instagram-private-api
